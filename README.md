# harti.at

Personal portfolio & project showcase by **Michael Hartmann** ([@Fibonacci00618](https://github.com/Fibonacci00618)).

**Live:** [https://harti.at](https://harti.at)

## Stack
- Vanilla HTML5 / CSS3 / JavaScript
- GSAP 3 (ScrollTrigger, animations)
- Three.js (WebGL particle field)

## Features
- Cinematic preloader with boot sequence
- Three.js particle field + wireframe icosahedron
- Lerp-based smooth scrolling
- Custom magnetic cursor
- Text split animations
- 3D card tilt effects
- Animated counters
- Keyword marquee
- Footer SVG wave animation
